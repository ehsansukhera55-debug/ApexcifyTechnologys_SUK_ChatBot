#  SUK Technologies FAQ Chatbot

An intelligent, NLP-powered FAQ chatbot designed for **SUK Technologies**. This assistant helps clients, employees, and applicants find information regarding company services, project status, technical support, and career opportunities.

##  Features
* **Natural Language Understanding:** Uses TF-IDF Vectorization and Cosine Similarity to understand user intent, even if the phrasing isn't an exact match.
* **Multi-Category Support:** Handles queries across 9+ departments (General, Accounts, Careers, Training, etc.).
* **Interactive Web UI:** Built with Streamlit for a clean, modern, and responsive chat experience.
* **Confidence Thresholding:** Only provides answers when the match is strong enough, ensuring professional and accurate communication.

##  Technical Stack
* **Language:** Python 3.x
* **Frontend:** [Streamlit](https://streamlit.io/)
* **NLP & Math:** [Scikit-learn](https://scikit-learn.org/) (TfidfVectorizer, Cosine Similarity)

## 📋 Categories Covered
- ✅ General Company Info
- ✅ Account & Login Assistance
- ✅ Product & Service Details
- ✅ Payment & Pricing Models
- ✅ Technical Support
- ✅ Real-time Project Status
- ✅ Company Policies
- ✅ Career & Internship Opportunities
- ✅ Training & Learning (SUK Academy)
